# VSCO Workspace - Pricing and Plans

All plans include full access to ALL Workspace features. Plans differ only in Users, Brands, and Bookable Event Types.

## Plan Tiers

### Solo Plan
- **Monthly**: ~$25/month
- **Annual**: $270/year (~$22.50/month, 10% discount)
- **Users**: 2
- **Brands**: 1
- **Bookable Event Type Templates**: 2-3
- All features included
- Client text messaging (U.S. & Canada)
- Phone, email, and chat support
- Free onboarding

### Boutique Plan
- **Monthly**: ~$35/month
- **Annual**: $378/year (~$31.50/month, 10% discount)
- **Users**: 6
- **Brands**: 4
- **Bookable Event Type Templates**: Unlimited
- All features included

### Studio Plan
- **Monthly**: ~$49.99/month
- **Annual**: $540/year (~$45/month, 10% discount)
- **Users**: 10 (additional users at $4.99/user)
- **Brands**: Unlimited
- **Bookable Event Type Templates**: Unlimited
- All features included

## Billing Options

- **Monthly**: Higher rates, auto-renewing, cancel anytime
- **Annual**: 10% discount, paid upfront, longer commitments save more
- Upgrades can be done anytime from Manage Subscription page

## Free Trial

- 14-30 days (standard), no credit card required
- 60 days via referral link

## Referral Program

- You get one month free credit when someone you refer subscribes
- Referred users get 60-day trial instead of standard

## Additional Costs (Not Included in Subscription)

- **SMS fees**: Standard third-party message delivery fees (~$0.01/segment via Quo)
- **Payment processing fees**: Stripe/Square fees apply on client payments
- No transaction fees from VSCO Workspace itself
