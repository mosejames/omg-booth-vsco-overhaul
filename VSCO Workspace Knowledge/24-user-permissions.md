# VSCO Workspace - User Permissions

## User Roles

**Location:** Settings > Add/Change Users

### Administrator
- Full access to all data
- Can make any changes
- Access to all settings areas
- Can view full Reports menu
- Can manage Scheduling sidebar (Bookable Events, Mini Sessions, Availability, Booking Sites)
- Can delete other user accounts

### Billing Setup
- Same permissions as Standard users
- Additionally can view and manage the Workspace subscription

### Standard
Cannot access these settings areas:
- Branding
- Mail Settings
- Calendar Settings
- Taxes & Currency
- Client Payment Methods
- Contact Forms
- Gallery Integrations
- Manage Subscription
- Automations
- API Integrations
- Early Access Betas

Cannot see:
- The entire Reports menu (but CAN see individual financial data on jobs and product/quote pricing)
- The Scheduling sidebar (no access to Bookable Events, Mini Sessions, Availability, or Booking Sites)

## Managing Users

- Add users at Settings > Add/Change Users via the "New User" button
- Users automatically receive an email invitation
- Must be an Admin to delete another user account
- Number of users depends on your plan tier:
  - Solo: 2 users
  - Boutique: 6 users
  - Studio: 10 users (additional at $4.99/user)
