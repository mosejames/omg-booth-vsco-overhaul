# VSCO Workspace - Calendar

## Google Calendar Integration

**Location:** Settings > Calendar Settings > New Google Integration

### Two Integration Options:

#### Option 1: Two-Way Real-Time Sync (Recommended)
- Changes in Google Calendar instantly reflect in Workspace and vice versa
- Pull events into Workspace by subscribing to Google Calendar feeds
- Push Workspace events to a new "Workspace" calendar feed in Google Calendar
- Event attendees see times, location, other attendees, and notes on their own calendar
- Google Meet links sync automatically -- if a synced event includes a Google Meet link, it appears in Workspace

#### Option 2: One-Way Calendar Feed Subscription
- Read-only, one-way feed
- Useful for privacy
- Good for restricting feeds to certain event types or job types
- More granular calendar feed filtering

## Apple Calendar (iCal)

### Mac:
- Subscribe to Workspace calendar feeds via File > New Calendar Subscription
- Customizable titles, colors, and notifications

### iPhone/iPad:
- Calendar subscriptions with customizable titles, colors, and notifications
- Subscribe to Workspace feeds in iOS Settings

---

## Calendar Setup

**Location:** Settings > Calendar Settings

### Calendar Feed Customization:
- **Kind:** Availability/Busy-Free, Only Confirmed Events, Only Tentative Events, All Events
- **Event Type:** Restrict to specific Event Types
- **Job Type:** Restrict to specific Job Types
- **Brand:** Restrict to specific Brands
- **Attendee:** Filter by specific attendees

### Options:
- Show availability as "busy" or "free"
- Restrict feeds to past 3 months or show entire history
- Subscribe to outside calendars within Workspace
- Create multiple feeds for different purposes

---

## Creating Calendar Feeds for Specific Filters

Create dedicated feeds showing only:
- Specific Event Types (e.g., only Sessions)
- Specific Job Types (e.g., only Weddings)
- Events with specific attendees (e.g., only your events)

Useful for team members who only need to see relevant events.
