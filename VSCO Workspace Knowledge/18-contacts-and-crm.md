# VSCO Workspace - Contacts and CRM

## Contact Profile

Each contact has its own profile in the Address Book.

### Overview Tab:
- Open Leads & Jobs
- Client Portal preview
- Open Financial Items (invoices)
- Future Interactions
- Related Contacts
- Possible Duplicate Contacts
- Edit Profile button
- Custom fields

### Additional Tabs:
- **Mail Tab** -- All emails for this contact
- **Financials Tab** -- All financial info across all leads & jobs
- **Notes Tab** -- Specific notes and file uploads
- **Leads & Jobs** -- All leads and jobs this contact is listed on

---

## Contact Relationships

- Link contacts for quick access to related people in dropdowns
- Create relationships from the job worksheet or Address Book profile
- Related contacts appear in dropdowns for quotes, emails, event attendees, etc.
- Useful for: family members, employers, venue contacts, other connections

---

## Contact Merge

- Merge duplicate People, Companies, or Locations
- Select two contacts > use Merge action button
- Prompted to resolve conflicts between entries
- Can also merge from the Possible Duplicate Contacts box on overview tab
- Limited to 2 contacts at a time, same type only

---

## Custom Tags

**Location:** Settings > Tags

- Create tags for labeling contacts (e.g., "VIP Clients," "Favorite Vendors")
- Apply from the Address Book profile or from any sidebar list
- Dropdown at top of each contact's profile to choose one or more tags
- Great for creating filtered searches based on criteria

---

## Importing Contacts

### Process:
1. Download the CSV template from the contact import page
2. Add your data into the template
3. Upload the file using "Upload File"
4. After import, contacts appear in "Recent Imports"
5. If something went wrong, use "Purge Imported Contacts" to delete and start over

### Requirements:
- Certain column headers must always be present (even if data fields are empty)
- All dates must use format: **YYYY-MM-DD**

---

## Custom Fields

**Location:** Settings > Custom Fields

- Create fields for Job Profile or Contact Profile
- Multiple field types available
- Each custom field automatically generates its own token
- Tokens can be used in emails, questionnaires, contracts, and automations
- Custom fields show up on the Job Overview in the Custom Fields section
- Can be referenced in automations via the Update Field Action

---

## Filters and Lists

### Address Book Filters:
- Filter and sort contacts
- Create custom filtered views (Custom Lists)
- Export filtered lists as CSV

### Sidebar Lists:
- Leads and Jobs sidebar lists include a Client Emails column
- Download via the download icon (top right)
- Options: download visible columns or all columns

---

## Exporting Contacts

- Download from Address Book or filtered lists as CSV
- Use for external email services (MailChimp, Flodesk, etc.)
- Create tailored lists using custom reports and tagging
