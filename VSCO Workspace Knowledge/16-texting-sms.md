# VSCO Workspace - Two-Way Texting / SMS

## Overview

VSCO Workspace enables two-way text messaging through a Quo (formerly OpenPhone) integration. Currently available in the U.S. and Canada only. 98% open rates on text messages.

## Setup

**Location:** Settings > Text Settings

### Steps:
1. Create a **Quo** (formerly OpenPhone) account
2. Choose a new phone number (search by area code/city) or port an existing number
3. Generate an **API key** in Quo's API settings
4. In Workspace: Settings > Text Settings > New Integration > Connect Quo
5. Paste your API key

### Cost:
- Each SMS segment (up to 160 characters) costs **$0.01** in Quo credit
- Standard third-party message delivery fees apply

### Regional Availability:
- Quo only allows creation of U.S. and Canada numbers

## Sending and Receiving

- All conversations sync between Quo and Workspace in ongoing message threads
- "Send Text" option appears anywhere you can compose a message
- Choose the recipient and select Continue
- Automate, personalize, and respond all from one place

## Automated Text Notifications

For bookable events, set up automatic text notifications:
- **Stale booking reminders** -- Remind clients about incomplete bookings
- **Confirmation notices** -- Confirm booked events via text
- **Reminder notices** -- Remind event attendees before their appointment

Select "add text notification" on the bookable event configuration page.
