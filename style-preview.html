Section 08

Master CSS Reference

One stylesheet that brands every client-facing page in VSCO Workspace — quotes, questionnaires, contracts, invoices, and the portal.

⚙
Paste into Settings > Branding > Advanced Design > Page CSS

Where this applies: This CSS is injected into every client-facing page VSCO Workspace renders — that includes quotes, questionnaires, contracts, invoices, and the client portal. You paste it once and it styles everything.

page-css.css 
/* ============================================
   OMG Booth — VSCO Workspace Page CSS
   Paste into: Settings > Branding > Advanced Design > Page CSS
   Applies to: Quotes, Questionnaires, Contracts,
               Invoices, and Client Portal
   ============================================ */

/* Google Font Import */
@import url('https://fonts.googleapis.com/css2?family=DM+Sans:wght@400;500;600;700&family=JetBrains+Mono:wght@400;500&display=swap');

/* Brand Variables */
:root {
  --omg-bg: #f5f4f1;
  --omg-white: #ffffff;
  --omg-dark: #1a1a1a;
  --omg-accent: #e8a849;
  --omg-accent-hover: #d4962f;
  --omg-muted: #888888;
  --omg-border: #e5e3de;
  --omg-font: 'DM Sans', sans-serif;
  --omg-mono: 'JetBrains Mono', monospace;
}

/* Base Styles */
body {
  font-family: var(--omg-font);
  background: var(--omg-bg);
  color: var(--omg-dark);
  line-height: 1.65;
  -webkit-font-smoothing: antialiased;
}

/* Headings */
h1, h2, h3, h4 {
  font-family: var(--omg-font);
  font-weight: 700;
  color: var(--omg-dark);
  letter-spacing: -0.02em;
}

/* Numbers, prices, invoice numbers */
.price, .total, .invoice-number, .amount,
[class*="number"], [class*="mono"] {
  font-family: var(--omg-mono);
}

/* Primary Buttons (CTA, Submit, Pay, Sign) */
button[type="submit"],
input[type="submit"],
.btn-primary, .cta-button,
a.button, .payment-button {
  background: var(--omg-accent);
  color: #ffffff;
  border: none;
  border-radius: 6px;
  padding: 0.75rem 1.5rem;
  font-family: var(--omg-font);
  font-weight: 600;
  font-size: 0.9rem;
  cursor: pointer;
  transition: background 0.2s, transform 0.15s;
  letter-spacing: -0.01em;
}
button[type="submit"]:hover,
input[type="submit"]:hover,
.btn-primary:hover, a.button:hover {
  background: var(--omg-accent-hover);
  transform: translateY(-1px);
}

/* Form Fields */
input[type="text"],
input[type="email"],
input[type="tel"],
input[type="date"],
input[type="number"],
textarea, select {
  font-family: var(--omg-font);
  border: 1.5px solid var(--omg-border);
  border-radius: 6px;
  padding: 0.65rem 0.9rem;
  font-size: 0.88rem;
  color: var(--omg-dark);
  background: var(--omg-white);
  transition: border-color 0.2s;
  outline: none;
}
input:focus, textarea:focus, select:focus {
  border-color: var(--omg-accent);
  box-shadow: 0 0 0 3px rgba(232,168,73,0.12);
}

/* Cards & Containers */
.card, .panel, .section-card,
.quote-card, .invoice-card {
  background: var(--omg-white);
  border-radius: 10px;
  box-shadow: 0 2px 12px rgba(0,0,0,0.06);
  border: none;
}

/* Links */
a {
  color: var(--omg-accent);
  transition: opacity 0.2s;
}
a:hover {
  opacity: 0.8;
}

/* Radio & Checkbox Accent */
input[type="radio"],
input[type="checkbox"] {
  accent-color: var(--omg-accent);
}

/* Status Badges */
.badge, .status-badge, .tag {
  font-family: var(--omg-font);
  font-size: 0.7rem;
  font-weight: 600;
  border-radius: 4px;
  padding: 0.15rem 0.55rem;
  text-transform: uppercase;
  letter-spacing: 0.04em;
}

/* Table Styling */
table {
  font-family: var(--omg-font);
  border-collapse: collapse;
  width: 100%;
}
th {
  font-size: 0.7rem;
  text-transform: uppercase;
  letter-spacing: 0.06em;
  color: var(--omg-muted);
  font-weight: 600;
  border-bottom: 2px solid #eee;
}
td {
  border-bottom: 1px solid #f0f0f0;
  padding: 0.7rem 0.8rem;
}

/* Smooth transitions */
*, *::before, *::after {
  transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
}

Note: VSCO Workspace uses its own class names internally. The selectors above target the most common patterns. After pasting, preview each page type (quote, form, contract, invoice, portal) and add any additional selectors needed for your specific templates. The :root variables make future color changes a one-line edit.
